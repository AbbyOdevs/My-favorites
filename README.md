Project Description

MyFavorites is a Windows Forms Application developed using C# and MySQL that allows users to manage their personal favorite items through a secure and user-friendly system. The project was created as the final requirement for the ITEC subject to demonstrate the application of programming concepts, database connectivity, and CRUD operations.

The system includes a Login and Registration Module where users can create accounts and securely access the application. After logging in, users can manage their records in the Favorites Management section. The application allows users to add, view, update, and delete information such as favorite colors, foods, and numbers. All records are stored in a MySQL database and displayed using a DataGridView for easier interaction and editing.

The project also includes input validation features to ensure that users provide complete and correct data before saving records. Duplicate entries are checked to maintain data consistency, and the system provides message prompts for successful operations and errors.

This project demonstrates the students’ understanding of:

C# Windows Forms programming
Database management using MySQL
CRUD operations (Create, Read, Update, Delete)
Event-driven programming
Data validation and user authentication
User interface design and navigation

Overall, MyFavorites serves as a simple yet functional database management system that highlights the practical use of programming and database integration in developing desktop applications.
